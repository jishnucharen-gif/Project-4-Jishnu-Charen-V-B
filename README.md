# Data Analytics Project 4 – Data Visualization

## Project Overview

This project is part of the **DecodeLabs Industrial Training Kit (Batch 2026)**. The objective is to create meaningful and interactive visualizations that help understand the dataset through graphical representation. The project focuses on presenting insights in a clear and visually appealing manner using Python visualization libraries.

---

## Objectives

- Create visual representations of data
- Identify trends and patterns
- Compare categories using charts
- Detect outliers using box plots
- Visualize correlations between variables
- Present business insights effectively

---

## Dataset Information

- **Dataset Name:** Cleaned_Dataset_for_Data_Analytics.xlsx
- **Total Records:** 1200
- **Total Columns:** 14

---

## Tools & Libraries Used

- Python
- Pandas
- Matplotlib
- Seaborn

---

## Visualizations Created

### 1. Histogram
- Distribution of numerical variables
- Example:
  - Total Price
  - Unit Price
  - Quantity

### 2. Bar Chart
- Orders by Status
- Sales by Category
- Payment Method Distribution

### 3. Box Plot
- Detect outliers in:
  - Unit Price
  - Total Price
  - Quantity

### 4. Scatter Plot
- Relationship between:
  - Unit Price vs Total Price
  - Quantity vs Total Price

### 5. Correlation Heatmap
- Shows relationships among numerical variables.

### 6. Pie Chart
- Payment Method Distribution
- Order Status Distribution

### 7. Line Chart
- Daily Sales Trend
- Monthly Sales Trend

---

## Key Insights

- Most orders fall within the mid-price range.
- A few high-value transactions are identified as outliers.
- Order statuses are fairly balanced.
- Sales vary across different product categories.
- Payment methods are distributed across multiple options.
- Numerical variables show moderate correlations.
